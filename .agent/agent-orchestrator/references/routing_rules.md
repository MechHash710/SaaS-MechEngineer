# Regras de Roteamento (Routing Rules)

O Orquestrador usa essas palavras-chave para descobrir qual agente deve receber uma tarefa listada no Kanban.

Se uma tarefa contiver palavras de MÚLTIPLOS domínios, você DEVE dividi-la em sub-tarefas antes de despachar, para garantir paralelismo limpo!
